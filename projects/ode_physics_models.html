Ode Physics Models

❮ ❯

ODEs were used to analyse the following:

  • Github - A bicycle model of a car - using ODEs to simulate turning stability
  • Github - The temperature of an egg placed in boiling water over time

As apart of Advanced Calculus, I used ODEs to model the temperature of an egg in boiling water and the stability of a bicycle model. The bicycle model is a simplified version of a car, which uses a two wheel model to represent the car's dynamics. The model is used to analyze the car's turning stability. The egg temperature model uses an ODE to simulate the temperature of an egg over time, which is placed in boiling water. More info regarding the projects can be found in the links above.
