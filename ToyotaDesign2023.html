Toyota Design Challenge 2023

  • Programmed in Python with OpenCV using Pytorch library
  • Used pretrained YOLO V5 model with dataset labeled with Roboflow

The Toyota Design Challenge in 2023 required that competitors be able to differentiate a hole, and a sticker stuck on one. Using YOLO V5 in combination with a dataset we were given, we developed a vision system to differentiate the two.
