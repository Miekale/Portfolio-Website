JestureBot - Make UofT 2023

❮ ❯
  • Used google hand recognition model with OpenCV to control car movements
  • Used logic level converter to drop voltage levels from 5.0V to 3.3V for Bluetooth module
  • Controlled 4 TT motors using L298N Motor Controller

This project uses OpenCV and Mediapipe's hand tracking model to determine the orientation of the person's hand. The software then sends a directional command to the RC car or instructs it to stop.
