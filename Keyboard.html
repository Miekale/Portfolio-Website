Keyboard from Scratch

❮ ❯
  • Developing Mechanical keyboard PCB in KiCAD using ATMega32u4 Microcontroller
  • Keymatrix build with N-key roll over and anti ghosting using diodes in the key matrix
  • Designed with USB-C 2.0 port operating at full speed

The next steps of this project will be to route the traces on the PCB in KiCad, design the case in mechanical CAD to be fabricated and set up the firmware on the microcontroller. To view access the pdf click here.
